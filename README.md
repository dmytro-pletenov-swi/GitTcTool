# GitTcTool
Get info about team's current PRs

## GetCurrentPR function
Return all team's current Pull Requests. 

### Params: 
`$author -a  string  optional`\
`$title  -t  string  optional`

### Examples:
`GetCurrentPR -author "plet" -title "oo-"`\
`GetCurrentPR -a ple`\
`GetCurrentPR`
